# Grida Design System v1.0

Grida is an open-source design system designed and created by ZiPIDA.

그리다는 지피다의 오픈소스 UI/UX 디자인 시스템입니다.

- 🤖 Auto Layout 2023

- 그리다 디자인 시스템의 패턴은 여러개의 컴포넌트로 구성되어 있습니다.
